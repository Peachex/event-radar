Tasks

The Tasks section allows you to manage and schedule predefined tasks that are already implemented on the server. These tasks are designed to execute specific logic (e.g., syncing the database) after a designated time interval, ensuring your system remains up-to-date without manual intervention.

Key features include:

  • Task Scheduling: Create schedules for predefined tasks and ensure they run automatically at specific times.
  • Task Execution: Trigger tasks to run at designated times, such as syncing the database or executing other repetitive processes.
  • Task Management: View, manage, and update task schedules to ensure everything runs as expected.

With the Tasks feature, you can easily create schedules for tasks, assign them to run at specified times, and ensure that important background operations (like database synchronization) are executed automatically.
