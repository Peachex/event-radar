Scheduler

The Scheduler is a powerful tool designed to manage the state of your tasks and control their execution schedule. It allows you to perform actions such as running tasks immediately, pausing, resuming, or deleting them. The Scheduler provides the flexibility to have multiple schedules for a single task

Key features of the Scheduler:

  • Task Management: Run tasks instantly, pause, resume, or delete tasks with ease.
  • Multiple Schedules: Create multiple schedules for a single task, allowing for better control over the timing of task executions.
  • Task Control: Pause a schedule for one task while running another, then resume the paused schedule whenever necessary.

This feature ensures that you can manage task execution with precision and control, making your task management process smooth and efficient.
